上海市陆生野生动物收容救护管理平台

基于《上海市陆生野生动物收容救护工作指南（试行）》的 H5 产品方案。覆盖接收→初检→移交→治疗检疫→安置全流程，解决工单跨角色流转断链、合规计时靠人记、年报手工汇总三大痛点。

产品目标

  • 三端 H5：救护工作台（手机）、管理后台（手机+PC 自适应）、市民端（手机）
  • 8 页极简线框图 + 完整 PRD（v4.0），含合规计时与预警规则
  • 动物分类：鸟类 / 兽类 / 爬行类，支撑统计分析与看板分组
≥90%工单系统使用率
≤5%24h 响应超时率
100%年报按时提交率

角色与端

救护员建单、填记录、移交、放归
监管员数据看板、预警中心、催办
审阅人年报审核通过 / 退回
市民提交求助、查进度

核心功能（P0）

  • 工单列表：待办 / 已办分栏，超时标红，20h 预警 / 24h 超时
  • 新建工单：6 字段 + 扫码接收市民求助
  • 工单处理：时间线 + 状态驱动操作按钮（检疫未满 30 天禁放归）
  • 数据看板：4 数字卡片 + 2 图表，市 / 区两级权限
  • 预警中心：超时 / 检疫 / 年报三类预警，一键催办

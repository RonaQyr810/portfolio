AI 视觉对话助手

打开摄像头与麦克风，让 AI 看见你的画面、听见你说的话，并给出自然的中文回复。实习期间独立发起的 AI 产品验证项目。

核心能力

  • 实时摄像头预览 + 连续语音识别（Web Speech API）
  • 用户说完一句话后自动截取画面，发送多模态大模型分析
  • 结合画面与对话历史回复，支持本地 / 云端语音朗读
  • 端云协同成本控制：按需截图、压缩去重、历史裁剪

技术栈

  • 前端：React + TypeScript + Vite
  • 后端：Node.js + Express，对接 OpenAI 多模态 API
  • 部署：Docker / Render 云端一键部署
查看 GitHub 源码 联系我体验

本项目需要 Node.js 后端与 API Key 才能完整运行，无法像静态页面一样直接嵌入。本地可双击项目目录中的 启动.bat 体验；云端部署见仓库内 docs/GITHUB_DEPLOY.md。

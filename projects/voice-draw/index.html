AI 语音绘图工具

纯语音控制的 Web 绘图应用——无需鼠标和键盘，通过中文语音指令在 Canvas 上完成创作。支持复合指令、撤销与画布清空。

核心能力

  • Web Speech API 中文语音识别，实时显示 interim / final 文本
  • 自然语言指令解析：形状、颜色、位置、尺寸与复合绘图
  • 麦克风音量可视化，自动检测权限与监听状态
  • 撤销 / 清空画布，指令执行反馈日志

常用指令示例

语音效果
开始监听 / 停止监听控制语音识别
画一个红色的圆绘制红色圆形
在中心画蓝色矩形 宽200 高100指定位置与尺寸
从左上到右下画一条线绘制直线
撤销 / 清空画布撤销或清空

技术栈

  • HTML5 Canvas + 原生 JavaScript（无框架依赖）
  • Web Speech API + 麦克风音量检测
  • 模块化：指令解析 / 绘图引擎 / 语音控制器

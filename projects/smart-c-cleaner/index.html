Smart C Cleaner

Windows 智能 C 盘清理工具，提供图形界面与命令行双模式。分类扫描临时文件、缓存与可安全删除项，深度清理项会二次确认，避免误删。

核心能力

  • Tkinter 图形界面：分类树形列表、勾选清理、进度条与磁盘空间概览
  • 安全清理 Safe 与深度清理 Deep 分级策略
  • 命令行模式：scan --deep / clean --deep -y
  • 管理员权限检测，部分系统目录需提权后清理

技术栈

  • Python 3 + Tkinter GUI
  • 模块化：scanner / clean_engine / categories
  • 可选 psutil 读取磁盘信息

本地运行

  • 源码位于桌面 开发项目/smart-c-cleaner/
  • 双击 启动图形界面.bat 或运行 python main.py
  • 依赖：pip install -r requirements.txt

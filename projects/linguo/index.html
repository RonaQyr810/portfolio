同声传译 Linguo

面向面对面双语对话的 AI 同声传译应用。支持双向实时语音识别、即时翻译、语音朗读与双语对照记录，Web 与 iOS（Capacitor）双端可用。

核心能力

  • 双向对话模式：自动识别说话语种，实时翻译到对方语言
  • 双引擎语音识别：Web Speech API + Capacitor 原生识别，支持中英文自动切换
  • 在线 / 离线翻译：OpenAI 云端翻译 + 本地 CTranslate2 离线兜底
  • 本地 Whisper ASR 后端：FastAPI 提供音频转写与 WebSocket 流式传译
  • 双语对照记录：支持复制、导出与清空会话
  • 文本翻译栏：手动输入文本即时翻译
🇨🇳 中文 🇺🇸 英语 🇯🇵 日语 🇰🇷 韩语 🇫🇷 法语 🇩🇪 德语 🇪🇸 西班牙语

技术栈

  • 前端：React + TypeScript + Vite + Tailwind CSS + Capacitor iOS
  • 后端：FastAPI + OpenAI API + 本地 Whisper 转写
  • 离线：CTranslate2 本地翻译引擎（无网络时可用）
联系我体验完整版

本项目为全栈应用，需启动 Python 后端并配置 API Key 才能完整运行，无法像静态页面一样直接嵌入。本地可在桌面 同声传译 目录启动前后端；iOS 版已通过 Capacitor 打包。

# 秦艺榕 · 个人主页

2026 届应届生作品集网站，包含 UI 设计、产品开发、视觉设计与视频动画等作品展示。

## 在线预览

部署后访问 Vercel 提供的地址（见仓库 Settings → Deployments）。

## 本地预览

双击 `start-site.bat`，或运行：

```bash
npx serve -l 8080
```

浏览器打开 http://127.0.0.1:8080
